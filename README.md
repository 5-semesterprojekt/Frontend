## Installation

Using [nvm](https://github.com/coreybutler/nvm-windows) is advised.

This project uses pnpm rather than npm, so this must be installed:

```
npm install -g pnpm
```

## Start

How to start up the project.

```
pnpm run dev
```

Go to: [http://127.0.0.1:3000/](http://127.0.0.1:3000/)
